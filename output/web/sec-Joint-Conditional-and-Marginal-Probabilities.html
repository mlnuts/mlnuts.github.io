    Section 1.5 Joint, Marginal, and Conditional Probabilities

    This section explores probability concepts for multiple random variables, essential for modeling relationships in data. We cover joint and marginal probabilities, covariance and correlation, conditional probability, Bayesâ€™ rule, and independence, with a focus on rigorous mathematical formulations and practical applications. Random variables are denoted by capital letters (e.g., \(X\)), with specific values in lowercase, sometimes with indices (e.g., \(x_i\)). Examples use a patient dataset to illustrate concepts, and Python visualizations reinforce intuition.
    ðŸ”—

    Subsection 1.5.1 Random Variables, Probabilities, and Expectations

    A random variable is a quantity whose value is determined by a random experiment, unknown until observed. We denote random variables by capital letters (e.g., \(X\)) and their values by lowercase with or without indices (e.g., \(x,\ x_i\)).
    ðŸ”—
    Random variables are:
    • Discrete: Takes values in a countable set (e.g., die outcomes \(\{1, 2, \dots, 6\}\)).
      ðŸ”—
    • Continuous: Takes values in a real interval (e.g., patient temperature).
      ðŸ”—
    ðŸ”—
    For a discrete random variable \(X\) with values \(x_i\) (\(i=1,\dots,N\)), the probability mass function (PMF) is:
    \begin{equation} P(X=x_i) = p_i, \quad i=1,\dots,N, \quad \sum_{i=1}^N p_i = 1.\tag{1.5.1} \end{equation}
    ðŸ”—
    The expectation of a function \(f(X)\) is:
    \begin{equation} \mathbb{E}[f(X)] = \sum_{i=1}^N f(x_i) p_i.\tag{1.5.2} \end{equation}
    The mean is:
    \begin{equation} \mathbb{E}[X] = \sum_{i=1}^N x_i p_i.\tag{1.5.3} \end{equation}
    ðŸ”—
    The variance measures spread:
    \begin{equation} \mathrm{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2.\tag{1.5.4} \end{equation}
    The standard deviation is \(\sigma_X = \sqrt{\mathrm{Var}(X)}\text{.}\)
    ðŸ”—

    Example 1.5.1. Die roll expectation and variance.

    For a fair six-sided die, \(P(X=k) = \frac{1}{6}\) for \(k=1,\dots,6\text{.}\) The mean is:
    \begin{equation*} \mathbb{E}[X] = \sum_{k=1}^6 k \cdot \frac{1}{6} = \frac{1+2+3+4+5+6}{6} = 3.5. \end{equation*}
    The second moment is:
    \begin{equation*} \mathbb{E}[X^2] = \sum_{k=1}^6 k^2 \cdot \frac{1}{6} = \frac{1^2 + 2^2 + \dots + 6^2}{6} = \frac{91}{6} \approx 15.1667. \end{equation*}
    The variance is:
    \begin{equation*} \mathrm{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 = \frac{91}{6} - (3.5)^2 = \frac{35}{12} \approx 2.9167. \end{equation*}
    ðŸ”—

    Subsection 1.5.2 Joint and Marginal Probabilities

    For two discrete random variables \(X\) and \(Y\text{,}\) the joint probability assigns probabilities to pairs \((x_i, y_j)\text{:}\)
    \begin{equation} P(X=x_i, Y=y_j) = p_{ij}, \quad \sum_i \sum_j p_{ij} = 1.\tag{1.5.5} \end{equation}
    ðŸ”—
    The marginal probability of \(X\) is obtained by summing over all values of \(Y\text{:}\)
    \begin{equation} P(X=x_i) = \sum_j P(X=x_i, Y=y_j) = \sum_j p_{ij}.\tag{1.5.6} \end{equation}
    Similarly, \(P(Y=y_j) = \sum_i p_{ij}\text{.}\)
    ðŸ”—

    Example 1.5.3. Patient disease and test results.

    Consider a dataset of 1,000 patients, with \(X\) indicating disease status (\(D\text{:}\) disease, \(N\text{:}\) no disease) and \(Y\) indicating test result (\(+\text{:}\) positive, \(-\text{:}\) negative). The joint counts are given in TableÂ 1.5.8. Joint probabilities are:
    \begin{equation*} P(D, +) = \frac{150}{1000} = 0.150, \quad P(D, -) = \frac{50}{1000} = 0.050, \end{equation*}
    \begin{equation*} P(N, +) = \frac{300}{1000} = 0.300, \quad P(N, -) = \frac{500}{1000} = 0.500. \end{equation*}
    Marginals are:
    \begin{equation*} P(D) = P(D, +) + P(D, -) = 0.150 + 0.050 = 0.200, \quad P(N) = 0.800, \end{equation*}
    \begin{equation*} P(+) = P(D, +) + P(N, +) = 0.150 + 0.300 = 0.450, \quad P(-) = 0.550. \end{equation*}
    ðŸ”—

    Subsection 1.5.3 Covariance and Correlation

    The covariance between random variables \(X\) and \(Y\) measures their joint variability:
    \begin{equation} \mathrm{Cov}(X,Y) = \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])] = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y].\tag{1.5.7} \end{equation}
    ðŸ”—
    The correlation normalizes covariance:
    \begin{equation} \mathrm{Corr}(X,Y) = \frac{\mathrm{Cov}(X,Y)}{\sigma_X \sigma_Y}, \quad \mathrm{Corr}(X,Y) \in [-1, 1],\tag{1.5.8} \end{equation}
    where \(\sigma_X = \sqrt{\mathrm{Var}(X)}\text{,}\) \(\sigma_Y = \sqrt{\mathrm{Var}(Y)}\text{.}\) Positive correlation means \(X\) and \(Y\) tend to increase together; negative means they move oppositely. Correlation does not imply causation.
    ðŸ”—

    Subsection 1.5.4 Conditional Probability

    The conditional probability of \(X=x_i\) given \(Y=y_j\) is:
    \begin{equation} P(X=x_i \mid Y=y_j) = \frac{P(X=x_i, Y=y_j)}{P(Y=y_j)}, \quad P(Y=y_j) > 0.\tag{1.5.9} \end{equation}
    By the product rule:
    \begin{equation} P(X=x_i, Y=y_j) = P(X=x_i \mid Y=y_j) P(Y=y_j).\tag{1.5.10} \end{equation}
    ðŸ”—
    Conditional probabilities form a distribution over \(X\) for fixed \(Y=y_j\text{,}\) summing to 1: \(\sum_i P(X=x_i \mid Y=y_j) = 1\text{.}\)
    ðŸ”—

    Example 1.5.7. Patient disease and test results.

    Using the patient dataset (TableÂ 1.5.8), compute all conditional probabilities: Joint probabilities:
    \begin{equation*} P(D, +) = 0.150, \quad P(D, -) = 0.050, \quad P(N, +) = 0.300, \quad P(N, -) = 0.500. \end{equation*}
    Marginals: \(P(D) = 0.200\text{,}\) \(P(N) = 0.800\text{,}\) \(P(+) = 0.450\text{,}\) \(P(-) = 0.550\text{.}\) Conditional probabilities:
    \begin{gather*} P(D \mid +) = \frac{0.150}{0.450} \approx 0.3333, \quad P(N \mid +) = \frac{0.300}{0.450} \approx 0.6667, \\ P(D \mid -) = \frac{0.050}{0.550} \approx 0.0909, \quad P(N \mid -) = \frac{0.500}{0.550} \approx 0.9091, \\ P(+ \mid D) = \frac{0.150}{0.200} = 0.750, \quad P(- \mid D) = \frac{0.050}{0.200} = 0.250, \\ P(+ \mid N) = \frac{0.300}{0.800} = 0.375, \quad P(- \mid N) = \frac{0.500}{0.800} = 0.625. \end{gather*}
    ðŸ”—

    Subsection 1.5.5 Bayesâ€™ Rule

    Bayesâ€™ theorem updates probabilities based on new evidence, derived from the product rule:
    \begin{equation} P(X=x_i \mid Y=y_j) = \frac{P(Y=y_j \mid X=x_i) P(X=x_i)}{P(Y=y_j)},\tag{1.5.11} \end{equation}
    where the denominator is the marginal probability:
    \begin{equation} P(Y=y_j) = \sum_k P(Y=y_j \mid X=x_k) P(X=x_k).\tag{1.5.12} \end{equation}
    ðŸ”—

    Example 1.5.10. Breast cancer screening.

    For a population with breast cancer prevalence \(P(C) = 0.01\text{,}\) a mammogram has:
    • Sensitivity: \(P(+ \mid C) = 0.80\text{.}\)
      ðŸ”—
    • Specificity: \(P(- \mid N) = 0.95\text{,}\) so \(P(+ \mid N) = 1 - 0.95 = 0.05\text{.}\)
      ðŸ”—
    Compute \(P(C \mid +)\text{.}\) First, the marginal probability:
    \begin{equation*} P(+) = P(+ \mid C)P(C) + P(+ \mid N)P(N) = 0.80 \cdot 0.01 + 0.05 \cdot 0.99 = 0.008 + 0.0495 = 0.0575. \end{equation*}
    Then:
    \begin{equation} P(C \mid +) = \frac{P(+ \mid C)P(C)}{P(+)} = \frac{0.008}{0.0575} \approx 0.1391.\tag{1.5.13} \end{equation}
    Thus, a positive test increases the probability of cancer from 1% to 13.91%.
    ðŸ”—

    Subsection 1.5.6 Continuous Random Variables

    For continuous random variables, probabilities are defined via density functions, extending discrete concepts to uncountable outcome spaces.
    ðŸ”—

    Subsubsection 1.5.6.1 Probability Density

    A continuous random variable \(X\) has a probability density function (PDF) \(f_X(x)\) such that the probability over an interval \([a, b]\) is:
    \begin{equation} P(a \leq X \leq b) = \int_a^b f_X(x) \, dx.\tag{1.5.14} \end{equation}
    The density normalizes:
    \begin{equation} \int_{-\infty}^{\infty} f_X(x) \, dx = 1.\tag{1.5.15} \end{equation}
    Expectation and variance are:
    \begin{equation} \mathbb{E}[X] = \int_{-\infty}^{\infty} x f_X(x) \, dx, \quad \mathrm{Var}(X) = \int_{-\infty}^{\infty} (x - \mathbb{E}[X])^2 f_X(x) \, dx.\tag{1.5.16} \end{equation}
    ðŸ”—
    For two continuous variables \(X\text{,}\) \(Y\text{,}\) the joint density \(f_{X,Y}(x, y)\) gives:
    \begin{equation} P(a \leq X \leq b, c \leq Y \leq d) = \int_c^d \int_a^b f_{X,Y}(x, y) \, dx \, dy.\tag{1.5.17} \end{equation}
    The marginal density of \(X\) is:
    \begin{equation} f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x, y) \, dy.\tag{1.5.18} \end{equation}
    ðŸ”—
    Example 1.5.12. Bivariate normal distribution.
    A bivariate normal distribution for \(X\text{,}\) \(Y\) with means \(\mu_X = \mu_Y = 0\text{,}\) variances \(\sigma_X^2 = \sigma_Y^2 = 1\text{,}\) and correlation \(\rho = 0.5\) has joint density:
    \begin{equation} f_{X,Y}(x, y) = \frac{1}{2\pi \sqrt{1 - \rho^2}} \exp\left(-\frac{x^2 - 2\rho xy + y^2}{2(1 - \rho^2)}\right).\tag{1.5.19} \end{equation}
    The marginal density of \(X\) is the standard normal: \(f_X(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}\text{.}\)
    ðŸ”—
    ðŸ”—
    # === Bivariate normal contour plot ===

    Subsubsection 1.5.6.2 Cumulative Distribution Function

    The cumulative distribution function (CDF) of a continuous random variable \(X\) is:
    \begin{equation} F_X(x) = P(X \leq x) = \int_{-\infty}^x f_X(t) \, dt.\tag{1.5.20} \end{equation}
    The density is the derivative: \(f_X(x) = \frac{d F_X}{dx}\text{,}\) where differentiable.
    ðŸ”—
    For a standard normal distribution (\(\mu = 0\text{,}\) \(\sigma = 1\)):
    \begin{equation} f_X(x) = \frac{1}{\sqrt{2\pi}} \exp\left(-\frac{x^2}{2}\right).\tag{1.5.21} \end{equation}
    Its CDF is \(F_X(x) = \Phi(x)\text{,}\) the standard normal CDF.
    ðŸ”—
    # === Standard normal PDF and CDF ===

    Subsection 1.5.7 Independent Random Variables

    Random variables \(X\) and \(Y\) are independent if their joint distribution factorizes:
    \begin{equation} P(X=x_i, Y=y_j) = P(X=x_i) P(Y=y_j), \quad \text{for all } i, j.\tag{1.5.22} \end{equation}
    For continuous variables:
    \begin{equation} f_{X,Y}(x, y) = f_X(x) f_Y(y).\tag{1.5.23} \end{equation}
    Equivalently, \(P(X=x_i \mid Y=y_j) = P(X=x_i)\) or \(f_{X|Y}(x \mid y) = f_X(x)\text{.}\)
    ðŸ”—
    Independence implies \(\mathrm{Cov}(X,Y) = 0\text{,}\) but zero covariance does not imply independence, except for jointly normal variables.
    ðŸ”—

    Example 1.5.17. Diagnosing disease with test and severity.

    Augment the patient dataset with symptom severity (\(S\text{,}\) normal, mean 5 for D, 3 for N, std 1.5). For a patient with a positive test (\(Y=+\)) and high severity (\(S \geq 4\)), compute \(P(D \mid +, S \geq 4)\) using conditional probabilities.

    import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

np.random.seed(42)

# Synthetic patient data

    data.extend([[1 if status == "D" else 0, test, s] for s in severity])

    if len(subset) > 0:
        p_d = np.mean(subset["Disease"])
        probs.append(p_d)
    else:
        probs.append(0)

# Plot
labels = ["Y=+, Sâ‰¥4", "Y=+, S<4", "Y=-, Sâ‰¥4", "Y=-, S<4"]
plt.figure(figsize=(8, 6))
plt.bar(labels, probs, color="blue")
