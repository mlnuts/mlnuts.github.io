    Section 1.4 Basic Probability

    Probability is essential in machine learning to model randomness in data and outcomes. Probability tells us how likely an event is to happen. An event in the context of probability is a description of an outcome of an experiment and is usually written as a set. For instance, if you toss a coin, the outcome ‚ÄúHead‚Äù is an event \(\{H\}\text{,}\) ‚ÄúTail‚Äù is an event \(\{T\}\text{,}\) and ‚ÄúEither Head Or Tail‚Äù is the event \(\{H,\ T\}\text{.}\) When you list multiple elements in an event, all it means that any of those outcomes will constitute this event having occurred.
    üîó
    Every trial of an experiment produces a binary answer yes/no to the question whether a particular event is the outcome of this trial. For instance, for a roll of a six-sided die suppose you got 2. That would mean any of the following events occurred: \(\{2\}\text{,}\) \(\{1,2\}\text{,}\) \(\{2,3\}\text{,}\) \(\{2,4\}\text{,}\) \(\{2,5\}\text{,}\) >m>\{2,6\}>/m>, \(\{1,2,3\}\text{,}\) \(\{1,2,4\}\text{,}\) etc.,i.e., all the events that contains \(2\) in it
    üîó
    An advantage of using sets to represent events is that you can use union and intersections, to build more complex events. For instance event \(\{H,\ T\} = \{H\}\cup \{T\}\text{,}\) which must always be true since it represents the event that in a toss, you got either \(H\) or \(T\) and those are the only two possibilities.
    üîó

    Subsection 1.4.1 Axiomatic View of Probability

    Andrey Kolmogorov laid the foundations of the subject of probability in 1933 with three fundamental axioms. Although these axioms are fundamental to understanding probability from basic mathematics, we will rarely work in the framework that we will study in this section. First we need definition of some terms in probability space.
    üîó
    Sample Space \(\Omega\text{:}\) Thesample space \(\Omega\) is the set of all unique outcomes in the experiment of interest. For instance, in the case of a single roll of a six-sided die, you can get only \(1, 2, 3, 4, 5, \text{or}, 6\text{.}\) Therefore, set \(\Omega = \{1, 2, 3, 4, 5, 6 \}\text{.}\) For a coin toss, \(\Omega = \{ H, T\}\text{,}\) where \(H\) is Heads and \(T\) is Tails.
    üîó
    Event Space \(F\text{:}\) Event Space \(F\) be the set of all subsets of \(\Omega\text{,}\) including the empty set, \(\varnothing = \{\}\text{.}\) Each event consists of none, one or any number of elementary outcomes listed in \(\Omega\) and represents all outcomes that would constitute that event if any of those was the result of that trial. Thus, if an event in a six-sided die roll is \(E = \{3, 5\}\) and if you roll the die and get a \(3\) or \(4\text{,}\) you will record that event \(E\) occurred and if neither showed up, i.e., you got \(1, 2, 4, \text{or}\ 6\text{,}\) then event \(E\) dis not occur in that trial.
    üîó
    If \(\Omega\) has \(N\) elements, then, \(F\) will have \(2^N\) elements. In the case of the six-sided die, the number of elements in \(F\) will be \(2^6 = 64\text{.}\) That‚Äôs too large to list in example here.
    üîó
    Let‚Äôs look at another experiment: tossing a coin with two possible outcomes in any single toss. So, \(\Omega = \{ H, T\}\text{,}\) where \(H\) is head and \(T\) is tail. The subsets will be
    \begin{equation*} F = \{ \{\}\equiv\varnothing, \{H\}, \{T\}, \{H,T\}\equiv\Omega \}\text{.} \end{equation*}
    All events are sets of elementary outcomes in the experiment! Thus, we can talk about union and intersection of events. For instance,
    \begin{equation*} \{H\} \cap \{T\} = \varnothing,\ \ \{H\} \cup \{T\} = \Omega. \end{equation*}
    üîó
    Notice that \(\varnothing\) and \(\Omega\) will always be in \(F\text{.}\) The empty set is included for completeness in mathematical calculations and \(\Omega\) here means either \(H\) or \(T\) event, \(\{H\}\) means an event in which it‚Äôs a head and \(\{T\}\) is an event of tail. Note: you can‚Äôt have an event in which a single toss will be head AND tail. That is why you don‚Äôt have that since \(H\) and \(T\) are unique exclusive outcomes.
    üîó
    What about two tosses of the same coin? Now, each element of the sample space will be outcomes of the two tosses. Writing the order one next to the other, we can list four possible outcomes:
    \begin{equation*} \Omega = \{HH, HT, TH, TT \} \end{equation*}
    This sample space has \(4\) elements. Therefore, it will have \(2^4 = 16\) subsets, each specifying an event.
    \begin{align*} F = \amp \{ \varnothing,\ \{HH\}, \{HT\}, \{TH\}, \{TT \}, \\ \amp \{HH, HT \}, \{HH, TH \}, \{HH, TT \}, \{HT, TH \}, \{HT, TT \}, \{TH, TT \}, \\ \amp \{HH, HT, TH \}, \{HH, HT, TT \}, \{HH, TH, TT \}, \{HT, TH, TT \}, \Omega \}. \end{align*}
    To complete the picture, we also need a Probability Measure \(P\text{:}\) that assigns a real number to each event \(E\) of the event space \(F\text{.}\) Let us work out \(P\) for the two toss of a single coin problem we have above.
    üîó
    If the coin was a fair coin, the four elementary outcomes in \(\Omega\) will all be equally likely, givin probability of the elementary events.
    \begin{equation*} P(\{HH\}) = P(\{HT\}) = P(\{TH\}) = P(\{TT \}) = \frac{1}{4}. \end{equation*}
    Probabilities of other events in.\(F\) are based on the third axiom given below since they are all (except for the \(\varnothing\) event) just union of the elementary events. Thus,
    \begin{align*} \amp P(\{HH, HT \}) = P(\{HH\}) + P(\{HT\}) = \frac{1}{4} + \frac{1}{4} = \frac{1}{2}.\\ \amp P(\{HH, HT, TH \} ) = \frac{3}{4}\\ \amp P(\Omega) = P(\{HH, HT, TH, TT \}) = \frac{4}{4} = 1. \end{align*}
    What about the probability of the empty event \(\varnothing \equiv \{\}\text{?}\) This event is complement of event \(\Omega\text{,}\) i., an event that says any of the possible outcomes of the experiment. So, the probability of the null event is zero.
    \begin{equation*} P(\varnothing) = P(\{\}) = 0. \end{equation*}
    üîó
    The trio \((\Omega, F, P)\) is called the Probability Space. The three axioms of the probability theory are:
    1. First Axiom: Non-negativity
      \begin{equation} P(E) \in \R,\ \ P(E) \ge 0 \text{ for all events } E \in F. \tag{1.4.1} \end{equation}
      üîó
      üîó
    2. Second Axiom: Normalization
      \begin{equation} P(\Omega) = 1.\tag{1.4.2} \end{equation}
      That is, in every trial, there is 100% certainty that one of the elementary events will be the outcome. Together with the first axiom, this implies that probability of any event has to be between \(0\) and \(1\text{,}\) inclusive.
      \begin{equation*} P(E) \in [0,1]\, \text{ for all events } E \in F. \end{equation*}
      üîó
      üîó
    3. Third Axiom: Additivity of probabilities of disjoint, i.e., mutually exclusive events.
      \begin{equation} P( E_1 \cup E_2 ) = P(E_1) + P(E_2) \text{ if } E_1 \cap E_2 = \varnothing. \tag{1.4.3} \end{equation}
      üîó
      üîó
    üîó
    From these axioms, we can show the following useful results.
    \begin{align} \amp P(\varnothing) = 0. \tag{1.4.4}\\ \amp P(E) + P(\text{not } E) = 1 \text{ for any event } E. \tag{1.4.5}\\ \amp P(E_1 \cup E_2) = P(E_1) + P(E_2) - P(E_1 \cap E_2). \tag{1.4.6} \end{align}
    The last one can be easily seen by drawing a ven diagram with events \(E_1\) and \(E_2\) overlapping as shown in Figure¬†1.4.1.
    üîó
    . A concrete example may help here. Suppose you have a six-sided die. Let \(E_1 = \{1, 3, 5\}\) and \(E_2 = \{3,5,6\}\text{.}\) The union of the two will be \(E_1 \cup E_2 = \{1,3,5,6\} \) and the intersection will be \(E_1 \cap E_2 = \{3,5\}\text{.}\) Note that the list of faces in \(E_1\) is saying that the roll will be either \(1\text{,}\) \(3\text{,}\) or \(5\text{.}\) Similarly for \(E_2\text{.}\) Then, we have
    \begin{align*} \amp P(E_1) = P('1') + P('3') + P('5'). \\ \amp P(E_2) = P('3') + P('5') + P('6'). \\ \amp P(E_1 \cap E_2) =P('3') + P('5') \end{align*}
    The probability of the union of the two sets is only
    \begin{equation*} P(E_1 \cup E_2) = P('1') + P('3') + P('5') + P('6'). \end{equation*}
    and the sum of the probabilities of the two events is
    \begin{equation*} P(E_1) + P(E_2) = P('1') +2\times P('3') + 2\times P('5') + P('6'). \end{equation*}
    Thus, we must subtract the probability of the intersection to remove the double counting.
    üîó

    Subsection 1.4.2 Three Types of Probabilities

    There are basically three ways of looking at probability:
    1. Theoretical or Classical Probability. It is based on making use of symmetry and calculating the possible outcomes in an experiment. For instance, if you have a six-sided die, which is not loaded to prefer one outcome or another, the chance of any one face showing up will be \(1/6\text{.}\) So, we can say that probability of any face as an outcome of a single roll is \(p = 1/6\text{.}\)
      üîó
      üîó
    2. Frequentist Probability. It is an empirical definition of probability by observation of repeated trials of the same experiment. Thus, in the case of a six-sided die, you will roll the die and observe how many times face with 1 dot showed up in how many rolls, each roll of the die being one trial of the experiment. Thus, if \(n_1\) of \(N\) trials had 1 dot face up. Then, we conclude that the ratio \(n_1/N\) is an approximation of the true probability \(p_1\) of face 1 in any roll. We say that, in the limit of infinitely many trials, we would get the "true" probability.
      \begin{equation} p_1 = \lim_{N\rightarrow \infty}\, \frac{n_1}{N}.\tag{1.4.7} \end{equation}
      We do not assume that probabilities of every face of the die is same, as we did in using the symmetry argument in the theoretical probability; we rely of the repeated trials to show us any differences among the faces.
      üîó
      üîó
    3. Bayesian Probability. This is also an empirical definition of probability. But, rather than give you one number for probability of an event, Bayesian gives you a probability distribution of the values of probability of the event. From that, you can work out the mean value, which you can use as one value for the probability of the event.
      üîó
      It is based on incorporating belief about the probability of an outcome BEFORE we even conduct the experiment and then updated this so-called prior assumption or bias with what we observe in the experiment. The updated belief is the posterior, and improved value of the probability.
      üîó
      Clearly, as we repeat the experiment infinitely many times, the effect of our initial belief would disappear and the answer will match the results of the frequentists‚Äô experiments. However, since we can never do infinite number of trials, the Bayesian gives an edge in cases where we have some information about the outcome even before we start the trials.
      üîó
      Example: This example is a little bit ahead of my presentation here as it requres a little bit of math to properly express how th Bayesian probability works. If you feel up to it, you can ahead and read on, but it‚Äôs okay to skip it for now.
      üîó
      In the case of a six-sided die, suppose we want to estimate the probability \(p_1\) for one-dot face up as we illustrated in the frequentist case above. First, we would need to choose a prior belief, i.e., a probability distribution for \(p_1\text{,}\) i.e., how likely is any value of \(p_1\) between its range of values, which will be from \(0\) to \(1\text{,}\) inclusive, \(0\le p_1 \le 1\text{.}\) Since, we do not know which value is right, we might decide that it could be 1/2 times it will be face up and 1/2 of the time it will be not face up (I know a fair die will be 1/6 times face up, but I want to show you how even a very off prior will eventually converge to the proper value). In such cases and our trial each time being either face up true or false ( which is a case of Bernoulli trials ), it is traditional to choose a beta distribution, which has two parameters \(\alpha\) and \(\beta\text{,}\) with \(\alpha=1\) and \(\beta = 1\text{.}\) Using symbol \(x\) for \(p_1\) and \(P(x)\text{,}\) probability density of \(p_1\text{,}\) we will write this as follows where \(0 \le x \le 1\text{.}\)
      \begin{equation} \text{P(x)} dx = \frac{1}{B(\alpha, \beta)}\, x^{\alpha - 1} (1-x)^{\beta - 1},\tag{1.4.8} \end{equation}
      where \(B(\alpha, \beta)\) is beta function. The mean value of beta distribution is an important result and can be easily found.
      \begin{equation} \langle x \rangle = \int_0^1\, x P(x) dx = \frac{\alpha}{\alpha + \beta}.\tag{1.4.9} \end{equation}
      Here, I have introduced physicists‚Äô notation for the mean of a quantity, \(\langle \cdots \rangle\text{.}\) Thus, by choosing \(B(1,1)\) as the prior distribution, we are assuming that somehow we suspect that \(p_1\) is close to \(1/2\text{.}\)
      \begin{equation} \langle p_1 \rangle = \frac{\alpha}{\alpha + \beta} = \frac{1}{1+1} = \frac{1}{2} = 0.5.\tag{1.4.10} \end{equation}
      So, we are basically, starting way off in our belief.
      üîó
      Just a side math info: Beta function is usually written in terms of factorial or Gamma function.
      \begin{equation*} B(\alpha, \beta) = \frac{\Gamma(\alpha) \Gamma(\beta)}{\Gamma(\alpha + \beta)}, \end{equation*}
      where, for integer arguments \(n\text{,}\)
      \begin{equation*} \Gamma(n) = (n-1)! = [ m! = (m)(m-1) \cdots 1 = m \times (m-1)!\ \text{with}\ 0! = 1.] \end{equation*}
      and in general,
      \begin{equation*} \Gamma(\alpha) = \int_0^\infty\, e^{-t}\,t^{\alpha -1}\, dt. \end{equation*}
      To hide all the mathematical details in our work below, we will, as is normally done, just express the probability \(P(p_1)\) by a simpler notation and represent Eq. (1.4.8)
      \begin{equation} P_1 \sim B(\alpha, \beta) = B(1,1),\tag{1.4.11} \end{equation}
      where instead of lower case variable name \(p_1\text{,}\) we use the notation of upper case \(P_1\text{.}\)
      üîó
      Let‚Äôs get back to our rolling experiment and see how our belief of the true value of \(p_1\) evolves with each roll‚Äôs result. Suppose we roll the die and observe that the up face is not one, then without showing you the calculations here, which will be done later in the chapter, we use Bayes rule, to be discussed later, to show that the probability distribution now shifts to \(B(1,2)\text{.}\)
      \begin{equation*} P_1 \sim B(1,1) \rightarrow \text{[Toss, No one]} \rightarrow \sim B(1,2) \Rightarrow \langle p_1 \rangle = \frac{1}{1+2} = \frac{1}{3}. \end{equation*}
      How did we go from distribution \(B(1,1)\) to \(B(1,2)\text{?}\) I used Bayesian theorem. We will not show the calculation here but differ to a later section.
      üîó
      Toss second time, let‚Äôs say the result is a one. Then our belief will be update with this new data to \(B(2,2)\text{.}\)
      \begin{equation*} P_1 \sim B(1,2) \rightarrow \text{[Toss, Yes one]} \rightarrow \sim B(2,2) \Rightarrow \langle p_1 \rangle = \frac{2}{4} = 0.5. \end{equation*}
      Toss again, say no one.
      \begin{equation*} P_1 \sim B(2,2) \rightarrow \text{[Toss, No one]} \rightarrow \sim B(2,3) \Rightarrow \langle p_1 \rangle = \frac{2}{2+3} = \frac{2}{5}. \end{equation*}
      We keep updating the probability distribution of \(p_1\text{.}\) At any point, we can take the expectation value of the the variable \(p_1\) in the current distribution to give us the "best current value" for \(p_1\text{.}\) Thus, after three trials above, we will say that \(p_1 = 0.4\text{.}\)
      üîó
      Suppose you continued rolling and you had the following next 7 trials:
      \begin{equation*} \text{1, not 1, not 1, not 1, not 1, 1, not 1}. \end{equation*}
      After these 10 trials in total, the distribution will be
      \begin{equation*} P_1 \sim B(4, 8) \Rightarrow \langle p_1 \rangle = \frac{4}{12} = \frac{1}{3}. \end{equation*}
      This is still far away from \(1/6\) that you would expect from a fair die, but you don‚Äôt know if the die was fair. So, empirical results are all you have to go by.
      üîó
      For the same rolling results, frequentists‚Äô probability will give us the following estimate:
      \begin{equation*} p_1(\text{frequentist}) = \frac{n_1}{N} = \frac{3}{10} = 0.3. \end{equation*}
      üîó
      They look similar. But, had you expected the die was fair, you would start with a better prior, with say \(B(1,5)\text{.}\) Then the 10 trials would update to
      \begin{equation*} P_1 \sim B(4, 12) \Rightarrow \langle p_1\rangle = \frac{4}{4+12} = \frac{1}{4} = 0.25. \end{equation*}
      It would have revealed if the die was not a fair die. It‚Äôs either not a fair die or we have rolled it too few times.
      üîó
