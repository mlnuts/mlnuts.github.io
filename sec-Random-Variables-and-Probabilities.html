    Section 1.5 Random Variables and Probabilities

    In this section we will present foundations of the calculational tools necessary for analytical work. First we will define random variables and then follow up with probability distributions of a single and then two random variables. The generalization of more than two random variables will be left for future sections.

    Subsection 1.5.1 Random Variables, Probabilities, and Expectations

    We will think of variables as something the is observed or measured by experiments. The outcome in any experiment is a real value of the variable. A variable whose value is uncertain or unpredictable or varies from trial to trial, even though measurement conditions haven’t changed, is called a random variable.
    We tend to use capital letter for the name of the variable and small letters for its values. Thus, for a variable \(X\text{,}\) the values will be denoted by \(x_1, x_2, \cdots,\) etc., or a generic value by just \(x\text{.}\) Sometimes we will use superscripts to denote values, \(x^{(1)},x^{(2)}, \cdots \text{.}\)
    An event will now refer to the outcome that in a particular trial, variable \(X\) has some value or a set of the possible values. Thus, \(X=\{x_1\}\) would be an event and so would be \(X=\{x_1, x_2\}\text{,}\) etc. In case of continuous values for \(X\text{,}\) and event may even be written as \(x_1 \lt X \le x_2\text{,}\) etc.
    A random variable can either continuous variable or a discrete variable. A continuous random variable takes values either in a finite segment of the real line or on an entire real line. For example, the price of a house (say, denoted by \(X\)) in the US dollars could be between \(10^4\) and \(10^8\text{.}\) We can say \(X \in [10^4, 10^8]\text{.}\)
    A discrete random variable takes values in a countable set e.g., die outcomes \(\in \{1, 2, \dots, 6\}\text{,}\) colors \(\in \{ \text{red}, \text{blue}, \text{green}, \text{magenta} \}\text{.}\) When a discrete random variable has abstract symbols or wrods as values, we seek numerical embeddings of them using real-valued vectors so that they can be fed into machine learning algorithms for processing.

    Subsection 1.5.2 Probability Mass Function

    For a discrete random variable \(X\) with values \(x_i\) (\(i=1,\dots,N\)), the probability mass function (PMF) assigns appropriate probabilities to events of all the unique values that the random variable can take. Thus, PMF of our example variable \(X\) will give all the \(p_i\) values in:
    \begin{equation} P(X=x_i) = p_i, \quad i=1,\dots,N, \quad \sum_{i=1}^N p_i = 1,\tag{1.5.1} \end{equation}
    where the last condition is important since it makes sure that probability values are properly normalized and all the unique events are included.

    Subsection 1.5.3 Expectation Values

    The expectation value of a function \(f(X)\text{,}\) which is also simply called expectation of \(f\text{,}\) is the weighted value of the function according to the probailities of each value of \(X\text{.}\)
    \begin{equation} \mathbb{E}[f(X)] = \sum_{i=1}^N f(x_i) p_i.\tag{1.5.2} \end{equation}
    The mean is just when the function is the variable itself.
    \begin{equation} \mathbb{E}[X] = \sum_{i=1}^N x_i p_i.\tag{1.5.3} \end{equation}
    In Physics, the expectation is commonly written as \(\langle f(X) \rangle\text{,}\) but in probability and statistics, the notation is \(\mathbb{E}[f(X)]\text{,}\) with \(f(X)\) in the bracket.
    The variance measures spread of distribution about the mean value. It is defined by
    \begin{equation} \mathrm{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2.\tag{1.5.4} \end{equation}

    Proof of the last step.

    Here, the second equality is easy to see by an explicit work, noting that \(\mathbb{E}[X]\) is just a number. Let’s denote it by letter \(\mu\text{.}\)
    \begin{align*} \mathbb{E}[(X - \mathbb{E}[X])^2]\amp = \sum_{i=1}^N (x_i - \mu)^2 p_i \\ \amp = \sum_{i=1}^N \,x_i^2 p_i + 2\mu \sum_{i=1}^N \,x_i^2 p_i - \mu^2 \sum_{i=1}^N \, p_i \\ \amp = \mathbb{E}[X^2] - 2\mu \times \mu +\mu^2 \times 1 \\ \amp = \mathbb{E}[X^2] - \mu^2 = \mathbb{E}[X^2] - (\mathbb{E}[X])^2.\quad \blacksquare \end{align*}
    The standard deviation is just square root of variance.
    \begin{equation*} \sigma_X = \sqrt{\mathrm{Var}(X)}\text{.} \end{equation*}

    Example 1.5.2. Die roll expectation and variance.

    For a fair six-sided die, \(P(X=k) = \frac{1}{6}\) for \(k=1,\dots,6\text{.}\) The mean is:
    \begin{equation*} \mathbb{E}[X] = \sum_{k=1}^6 k \cdot \frac{1}{6} = \frac{1+2+3+4+5+6}{6} = 3.5. \end{equation*}
    The second moment is:
    \begin{equation*} \mathbb{E}[X^2] = \sum_{k=1}^6 k^2 \cdot \frac{1}{6} = \frac{1^2 + 2^2 + \dots + 6^2}{6} = \frac{91}{6} \approx 15.1667. \end{equation*}
    The variance is:
    \begin{equation*} \mathrm{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 = \frac{91}{6} - (3.5)^2 = \frac{35}{12} \approx 2.9167. \end{equation*}

    Subsection 1.5.4 Probability Density of Continuous Variables

    Probability distribution of a continuous variable cannot be given by a PMF since there are uncountably infinitely many unique values in the sample space \(\Omega\text{,}\) which is taken to be the entire real line.
    \begin{equation*} \Omega = \mathbb{R}. \end{equation*}
    If we have an \(X\) that is non-zero only in an interval, we can extend this \(X\) to the entire real line by just stating that the probability for all other values be zero.
    Instead of PMF, it turns out that the best we can do in the case of continuous variables is to define probability within an infinitesimal interval around some value \(x\) when we want to know about the distribution of probabilities “at” \(x\text{.}\) Thus, we nead probability per unit interval, i.e., probability density function (PDF) at \(x\text{,}\) which we will denote by \(\rho(x)\text{.}\)
    \begin{equation} P(x - dx/2 \le X \le x + dx/2 ) = \rho(x)\, dx \equiv P(x \le X \le x + dx ).\tag{1.5.5} \end{equation}
    Events are then defined by an interval in which you will find the value of the random variable \(X\text{.}\) Thus, the probability of the event that \(X \in [a, b]\) will be
    \begin{equation*} P(a\le X \le b ) = \int_a^b\, \rho(x)\,dx. \end{equation*}
    Normalization requires that this integral over all values of \(x\) be \(1\text{.}\)
    \begin{equation} P(x \in \mathbb{R}) = \int_{-\infty}^\infty\, \rho(x)\,dx = 1.\tag{1.5.6} \end{equation}
    A very useful probability density is a uniform value between, say \(x=a\) and \(x = b \gt a\text{.}\) Due to normalization, the constant value of \(\rho(x) = \frac{1}{b-a}\text{.}\)
    \begin{equation*} \rho(x) = \begin{cases} 0 \amp x \lt \\ \frac{1}{b-a} \amp a \le x \le b\\ 0 \amp x \gt b \end{cases} \end{equation*}
    You can easily check that it is properly normalized.
    \begin{equation*} \int_0^{\infty}\rho(x) dx = 0 + \int_a^b \frac{1}{b-a}\,dx + 0 = 1. \end{equation*}
    Another probability density of extreme importance is the Gaussian density function centered about zero and having unit variance. This Gaussian is called normal density.
    \begin{equation*} \rho = \frac{1}{ \sqrt{2\pi} }\,e^{-x^2/2}. \end{equation*}
    Although, it is more difficult to show but the integral of a Gaussian with the factor as shown does give you the correct normaliztion.
    \begin{equation*} \frac{1}{ \sqrt{2\pi} }\,\int_{-\infty}^{\infty}\,e^{-x^2/2}\, dx = 1. \end{equation*}

    Proof.

    Proof: Let \(I\) denote the following integral.
    \begin{equation*} I = \int_{-\infty}^{\infty}\,e^{-x^2/2}\, dx. \end{equation*}
    We need to show that \(I = \sqrt{2\pi}\text{.}\) We look at \(I^2\) and write the second \(I\) using another dummy symbol \(y\text{.}\)
    \begin{equation*} I \times I = \left[ \int_{-\infty}^{\infty}\,e^{-x^2/2}\, dx \right] \times \left[ \int_{-\infty}^{\infty}\,e^{-y^2/2}\, dy \right] \end{equation*}
    Now, combine the two integrals and think of them as integral in the \(xy\)-plane.
    \begin{equation*} I^2 = \int_{-\infty}^{\infty}\,e^{-(x^2 + y^2)/2}\, dxdy. \end{equation*}
    Now, change variable to polar coordinates \(r,\theta\) with
    \begin{equation*} dx dy = r dr d\theta. \quad \end{equation*}
    With \(x^2 + y^2 = r^2\text{,}\) the integral over \(\theta\) separates out and gives \(2\pi\) leaving only integral over \(r\text{,}\) which goes from \(0\) to \(\infty\text{.}\)
    \begin{equation*} I^2 = 2\pi \int_{0}^{\infty}\,e^{-r/2}\, r dr. \end{equation*}
    The integral over \(r\) is easily done by substitution \(u=r^2\text{.}\) Show that the value of this integral is just \(1\text{.}\)
    \begin{equation*} I ^2 = 2\pi. \end{equation*}
    Since \(I \gt 0\text{,}\) only positive root is applicable. Hence
    \begin{equation*} I = \sqrt{2\pi}\qquad \blacksquare \end{equation*}
    Expectation values of functions continuous random variables are also weighted values of the functions. But, here you need to do the integral.
    \begin{equation} \mathbb{E}[f(X)] = \int_{-\infty}^{\infty}\,f(x)\,\rho(x)\,dx.\tag{1.5.7} \end{equation}
    Accordingly, mean, variance, and standard deviations will be
    \begin{align} \amp \mu = \int_{-\infty}^{\infty}\,x\,\rho(x)\,dx. \tag{1.5.8}\\ \amp \mathrm{Var}(X) = \int_{-\infty}^{\infty}\,x^2\,\rho(x)\,dx - \mu^2 \equiv \sigma^2.\tag{1.5.9}\\ \amp\sigma = \sqrt{ \mathrm{Var}(X) } \tag{1.5.10} \end{align}
    We will study examples of various distribution functions in the next section.

    Subsection 1.5.5 Cumulative Distribution Function (CDF)

    For continuous distribution functions, we can find the probability of an event that the outcome of an observation on the random variable \(X\) will be less than or equal to some real number \(x\) by the following integral.
    \begin{equation} P( X \le x) = \int_{-\infty}^x\, \rho(x')\, dx'.\tag{1.5.11} \end{equation}
    It’s important to note that on the right side the \(x\) is some definite fixed value and that is why I am using \(x'\) as the dummy variable for the integral. You could use \(t\) or \(y\) or any other symbol for the dummy variable. Try not to confuse the dummy variable with the value variable. The value \(x\) can be any value on the real line.
    Probability in Eq. (1.5.11) is called cumulative distribution function (CDF) corresponding to that probability density \(\rho\text{.}\) We typically use symbol \(F\) for the CDF. It is clearly a function of the value variable \(x\text{.}\)
    \begin{equation} F (x) = \int_{-\infty}^x\, \rho(x')\, dx' = P( X \le x).\tag{1.5.12} \end{equation}
    The integral-derivative relation in the fundamental theorem of Calculus immediately gives us the following relation between \(F\) and \(\rho\text{.}\)
    \begin{equation} \rho(x) = \frac{dF}{dx}.\tag{1.5.13} \end{equation}
    From the definition in Eq. (1.5.12) it is clear that when \(x=\infty\text{,}\) then we would be integrating the PDF over the entire real line, which by normalization should be \(1\text{.}\)
    \begin{equation*} F(+\infty) = 1. \end{equation*}
    Similarly, when \(x=-\infty\text{,}\) the integral should be zero.
    \begin{equation*} F(-\infty) = 0. \end{equation*}
    Thus over the real line, CDF goes from \(0\) at \(x=-\infty\) to \(1\) at \(x=+\infty\text{.}\) Figure 1.5.3 shows this behavior of CDF of two distributions. Notice that the shape of the CDF’s are different for different distribution functions and carry the information of the probabilities of events in different intervals.
